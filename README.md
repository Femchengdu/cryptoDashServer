# Basic Express Server For React SPA

This is a basic express server to load a react crypto SPA

## Getting Started

To run the application, follow the instructions below

### Installing

```
cd cryptoDashServer
```

And run

```
npm install
```

Then run

```
node app.js
```

## Built With

*  ReactJs
*  ExpressJs
